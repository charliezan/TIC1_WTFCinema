# WTF Cinema

Sistema web de gestión y venta de entradas para cine y teatro.

WTF Cinema es una plataforma diseñada para digitalizar y centralizar la administración de funciones, salas, películas/eventos y venta de entradas en un entorno moderno e intuitivo.

---

## Funcionalidades

- Gestión de películas / eventos / funciones  
- Administración de salas y horarios  
- Venta y reserva de entradas  
- Panel administrativo para gestión del sistema  
- Interfaz amigable para usuarios y administradores  

---

## Tecnologías Utilizadas

- HTML / CSS / JavaScript 
- Java Spring Boot
- PostgreSQL

---

## Objetivo del Proyecto

Desarrollar una solución digital para optimizar la gestión operativa de cines y teatros, mejorando tanto la experiencia del usuario final como la administración interna.

---

## Contexto

Proyecto académico desarrollado como parte de la materia TIC / desarrollo web.
